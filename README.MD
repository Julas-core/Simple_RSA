Simple RSA — Learning Project
=============================

Introduction
------------

A very small, beginner-friendly RSA demo written in plain Python.  
Simple code, no advanced tricks — ideal for beginners and for learning how RSA works.

------------------------------------------------------------

What it does
------------

- Generate n and phi using two prime numbers  
- List possible values for e  
- Generate private key d  
- Encrypt messages  
- Decrypt messages  

------------------------------------------------------------

How to Run
----------

Open terminal in this folder and run:

python RSA_Encryption.py

------------------------------------------------------------

Example
-------

p = 17  
q = 11  
e = 7  

Encrypt: HI  
Output: 30 61  

Decrypt: 30 61  
Output: HI
